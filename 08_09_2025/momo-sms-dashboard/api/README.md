# Momo SMS Dashboard API

This is a Flask-based API for managing mobile money transactions. It supports CRUD operations and HTTP Basic Authentication.

## Requirements

- Python 3.13+
- Flask 3.1.2

cd <repo-folder>

2.**Create a python environment
